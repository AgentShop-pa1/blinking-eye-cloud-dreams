# 👁️ Blinking Eye - Cloud Dreams

A beautiful, animated web application featuring a realistic blinking eye with a dreamy cloud theme.

## Features

- 🌤️ **Cloud Theme**: Soft, airy design with floating cloud animations
- 👁️ **Realistic Eye**: Detailed eye with iris, pupil, and eyelashes  
- ✨ **Mouse Tracking**: Eye follows cursor movement naturally
- 💫 **Smooth Animations**: Gentle blinking, breathing effects, and floating particles
- 📱 **Responsive Design**: Works perfectly on all devices
- 🎨 **Dynamic Colors**: Iris color changes based on time of day
- ⚡ **Interactive**: Click the eye for instant blink

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Fonts**: Google Fonts (Comfortaa, Inter)
- **Deployment**: Railway + Express.js
- **Styling**: CSS Grid, Flexbox, CSS Animations

## Design Features

- Soft color palette inspired by clouds and sky
- Professional typography with Google Fonts
- Smooth transitions and hover effects
- Floating particle effects for ambiance
- Adaptive iris colors for different times of day

## Local Development

```bash
npm install
npm start
```

Visit `http://localhost:3000`
